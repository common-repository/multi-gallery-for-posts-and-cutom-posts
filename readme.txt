== Description ==
 
 This plugin will enable more images for each post types. you can change settings from media->Multi Gallery
 
 
== Installation ==
 
1. Upload `Multi_gallery` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto Multi Gallery settings under media menu
4. change the settings where  you want multi imags to upload
5. Go to post type where you want and uplad images as many as you want
6. write code to display images using 'multi_image_gallery' as meta key
